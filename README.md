# Animated Footer with Custom 404 Page

A responsive website footer component built using HTML and CSS, featuring smooth animations, social media navigation links, and a custom-designed 404 error page for unavailable destinations.

## Features

* Responsive footer layout
* CSS animations and transition effects
* Social media navigation section
* Custom animated 404 error page
* Mobile-friendly design
* Flexbox and CSS Grid based layout
* Consistent UI and visual styling

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

## Project Highlights

* Designed a modern footer component with smooth hover interactions and animated elements
* Implemented responsive layouts for desktop, tablet, and mobile devices
* Connected footer navigation links to a custom-built 404 page to simulate broken or unavailable destinations
* Created a visually engaging 404 page using pure CSS animations and custom styling
* Focused on clean code structure and reusable frontend design practices

## Learning Outcomes

Through this project, I practiced:

* Responsive web design
* CSS animations and transitions
* Flexbox and Grid layouts
* UI component development
* Error page design and navigation handling
* Frontend code organization
